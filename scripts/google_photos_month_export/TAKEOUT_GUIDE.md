# Google フォトを Takeout でバックアップする（推奨運用）

**Google Takeout** は Google 公式のエクスポート手段です。**OAuth や開発者向け API は不要**です。  
2025年以降のポリシーでは、**ライブラリ全体を月単位で自動取得する用途は Takeout が現実的**です。

## 手順の概要

1. ブラウザで **[Google Takeout](https://takeout.google.com/)** を開く。
2. **「すべてのデータが含まれています」** をクリックし、いったん **すべてのチェックを外す**。
3. 一覧から **「Google フォト」だけにチェック** を入れる（他は不要なら外す）。
4. 画面下の **「次のステップ」** に進む。
5. **ファイルの種類・サイズ**  
   - 形式: 多くの場合 **`.zip` のまま**で問題ありません。  
   - 1 ファイルの最大サイズ: 大きいライブラリなら **複数に分割**（例: 2GB や 10GB 単位）を選ぶとダウンロードしやすいです。
6. **書き出しの実行** を押す。完了まで **数時間〜数日** かかることがあります（メール通知が来ます）。
7. 表示されたリンクから **ZIP をダウンロード**（期限あり。期限内に取り込む）。
8. Mac では **ダブルクリックで解凍**。フォルダ構成はだいたい次のようになります。  
   `Takeout/Google Photos/` 以下に、**アルバム名や日付に基づくサブフォルダ**に写真・動画が入ります。

## 「指定の月だけ」に近づけたい場合

Takeout の UI は **「2024年3月だけ」という細かい日付指定には向いていません**。次のような運用が現実的です。

- **期間で絞れるオプションが出る場合**は、それに合わせて書き出しを分ける。  
- **まとめて書き出し**したあと、Mac 側で **撮影日でソート・スマートアルバム** や、**写真.app / Finder** で整理する。  
- **毎月同じ作業にしたい**場合は、カレンダーに「Takeout 実行日」を決め、分割サイズを小さめにしてダウンロード負担を減らす。

## 注意

- 書き出しリンクには **有効期限**があります。メールが来たら早めにダウンロードしてください。
- **RAW や大容量動画**が多いと、ZIP 分割数が増えます。
- **共有アルバム**や **パートナー共有**の扱いは、Takeout の説明と実際のチェック内容に依存します。必要なら Google フォト側で対象を確認してください。

## 関連ファイル

- **手早くボタンから開きたい場合:** 同フォルダの **`Launch-Takeout-Guide.command`**（または **`Launch-Photos-Export.command`**）で、簡易ヘルパー画面が開きます。
