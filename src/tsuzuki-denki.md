## 事業概要

ネットワークシステムおよび情報システムの**設計・開発・施工・保守**を主力事業とするシステムインテグレータ（SIer）。

### 主な事業セグメント

| セグメント | 内容 |
|---|---|
| **ネットワークソリューションサービス** | 通信インフラ・業務システムの設計・構築・運用保守、クラウドサービス、DX支援 |
| **電子デバイス** | 電子部品・デバイスの販売（※2024年に当該事業子会社の株式を譲渡） |

## 経営理念（パーパス＆バリューズ）

2022年の創業90周年を機に理念体系を刷新。

### パーパス（私たちの価値とあり方）

> **「人と知と技術」で、可能性に満ちた"余白"をともに。**

- **人と知と技術** — 価値の源泉であり、独自のアイデンティティ
- **可能性に満ちた"余白"** — お客さまや社会に対して生み出し続ける価値
- **ともに** — ステークホルダーと伴走・協働する企業姿勢

### バリューズ（7つの行動指針）

`向き合う` `築く` `つなぐ` `挑む` `楽しむ` `支援する` `やり抜く`

> パーパスは全従業員へのアンケート（回収率80%）と多様なメンバーによるワークショップを経て、ボトムアップで制定された。

## 沿革

### 創業期（1930〜1960年代）

| 年 | 出来事 |
|----|--------|
| 1932 | 名古屋で**都築商店**として創業（電話を中心とする電気通信設備工事） |
| 1941 | 株式会社に改組、**都築電話工業株式会社**となる |
| 1945 | 空襲により本社全焼 → 本社を東京（大田区）に移転 |
| 1958 | **電算機・関連機器の販売**を開始（IT事業の原点） |
| 1963 | 株式を日本証券業協会に店頭登録 |

### 成長期（1970〜1990年代）

| 年 | 出来事 |
|----|--------|
| 1970 | 都築ソフトウェア株式会社 設立 |
| 1981 | **OA事業部**を設置 |
| 1986 | 東京証券取引所 市場第二部 **上場** |
| 1988 | 通産大臣認定の**システムインテグレータ企業**となる |
| 1991 | **都築電気株式会社**に社名変更 |
| 1994 | シンガポールに現地法人設立（海外展開開始） |

### 現代（2000年代〜）

| 年 | 出来事 |
|----|--------|
| 2001 | ISO9001 認証取得 |
| 2018 | 東証の業種区分を「卸売業」→「**情報・通信業**」に変更 |
| 2020 | 東証市場第一部に指定 / 株式会社コムデザインを子会社化 |
| 2021 | 電子デバイス事業を分社化（都築エンベデッドソリューションズ設立） |
| 2022 | **東証プライム銘柄**に移行 / **DX認定事業者**に認定 / パーパス制定・経営理念刷新 |
| 2024 | 電子デバイス系子会社の株式を株式会社レスターホールディングスに譲渡 |
