# バディカフェニックス（BUDDICA PHOENIX BODY）洗車・施工順｜図解用原本

- **作成日**: 2026-04-16
- **目的**: 一式を揃えた際の「いつ・何をするか」を第三者向け資料として図解化するための構造化メモ
- **対象**: BUDDICA PHOENIX BODY（バディカ×ながら洗車）ボディ用ライン（シャンプー・ベース・コーティング）を中心とした洗車セッション

---

## 要点（WHAT）

| 区分 | 内容 |
|------|------|
| 製品の役割 | シャンプー＝洗浄／ベース＝下地・充填（水シミ・洗車傷の見た目改善）／コーティング＝艶・撥水のトップ層 |
| 同一セッション内の基本順 | **シャンプー洗車 → すすぎ → PHOENIX BASE（濡れボディ）→ PHOENIX COATING（湿式）** |
| キーワード | ベース・コーティングは**湿式**（水滴が残る状態）で施工する説明が多い。コーティングは**少量・素早い拭き上げ**でムラ防止 |

---

## フル施工の日（HOW・手順）

1. **ホイール・下回り**（別製品の場合）— ボディ洗浄より先に済ませ、汚水がボディに乗らないようにするのが一般的
2. **流水・予洗い**— 砂・泥を流してからボディに触れる
3. **PHOENIX SHAMPOO で洗車**— ボディの汚れを落とす
4. **（任意）鉄粉除去・粘土**— 塗装がザラつく場合。説明記事ではこの段階での投入が推奨されることがある
5. **すすぎ**— シャンプー・ケミカル残りを流す
6. **PHOENIX BASE**— 洗車直後の**濡れたボディ**でパネル単位に塗り広げ、余剰を水または固く絞った濡れクロスで処理
7. **PHOENIX COATING**— **湿式**。少量スプレー→濡れクロスで伸ばす→**別クロス**で拭き上げ。施工後しばらくは水濡れを避ける説明が多い
8. **ガラス・樹脂・タイヤ等**— ボディのベース／コーティング後に、別製品なら仕上げるのが一般的（製品ラベルで部位を確認）

### 注意（RESULT / リスク回避）

- **炎天下・ボディが熱い状態**は液の乾きが早くムラの原因になりやすい
- コーティング施工後は**数時間〜24時間は雨・洗車を避ける**などの説明がある
- コーティングのムラリカバリーとして**ベースでリセット**できる、という解説がある（各製品の最新表示に従うこと）

---

## 日常メンテ（頻度のイメージ）

| 状況 | 想定アクション |
|------|----------------|
| 普段の洗車 | PHOENIX SHAMPOO を中心に洗浄 |
| 撥水・艶が弱い | PHOENIX COATING を追加 |
| 水シミ・小傷・くすみが気になる | PHOENIX BASE で下地をやり直す（解説では数ヶ月〜半年に一度などの言及あり） |

※頻度は保管環境・走行距離で変わる。**メーカー・販売元の最新の表示を優先**すること。
